# GreenMedical

Rediseño web institucional para Green Medical, laboratorio chileno de suplementos alimentarios y productos naturales.

## Objetivo
Modernizar la presencia digital de una empresa real mediante una interfaz más actual, catálogo interactivo, secciones institucionales y canales de contacto claros.

## Tecnologías
- HTML5
- CSS3
- JavaScript
- Bootstrap Icons
- Google Fonts

## Características
- Diseño responsive
- Catálogo dinámico desde archivo JavaScript generado a partir de Excel
- Formulario con validación
- Modal de contacto
- Carrusel institucional
- Video institucional
- Integración con WhatsApp, correo, teléfono y Google Maps
- Animaciones y microinteracciones
